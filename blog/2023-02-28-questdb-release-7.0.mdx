We have released QuestDB 7.0! The main feature,
[Write Ahead Log (WAL)](/docs/concept/write-ahead-log/), allows concurrent data
ingestion across all interfaces and improves the data ingestion rate. WAL is
also the first step for achieving high availability and replication, both are on our
roadmap. [Read more](https://github.com/questdb/questdb/releases/tag/7.0.1)
