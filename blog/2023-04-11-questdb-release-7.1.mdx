We have released QuestDB 7.1! This release provides WAL storage optimization,
improved Web Console UI. It also implements SQL
[SHOW PARTITIONS](/docs/reference/sql/show/) and
[table_partitions](/docs/reference/function/meta/#table_partitions) for
partition setting informations.
